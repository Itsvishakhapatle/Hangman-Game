# Hangman-Game
Hangman Game by using Python 
A simple and interactive **command-line Hangman game** built using Python.  
This project demonstrates core programming concepts like loops, conditionals, lists, and user input handling.

---

## 🚀 Features

- Random word selection
- User-friendly command-line interface
- Limited attempts (game challenge)
- Real-time word guessing display
- Win/Loss condition handling

---

## 🛠️ Technologies Used

- Python
- Random Module
